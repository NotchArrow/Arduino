# Piano
This Arduino piano project used programmable keyboard buttons and the LCD screen to allow a variety of piano melodies. The notes for each button can be set individually, allowing for songs with up to 7 different notes. However, the Arduino `tone()` function and the buzzer did not easily support chords. The project was eventually adapted to include the [Clock](https://github.com/NotchArrow/Arduino/tree/main/Clock) project as one of the screen settings, and a temperature display was added into the screen, supporting Fahrenheit and Celcius.
