# Clock
This was one of my first Arduino projects, so the code is fairly messy and I didn't have a lot of breadboarding tools and materials to work with. It is a fully functional digital clock with AM & PM, relying on the internal Arduino `millis()` timer, which is unfortunately about 0.15% slow on average. I eventually added a temperature display and rebuilt the clock as a screen in my [Piano](https://github.com/NotchArrow/Arduino/tree/main/Piano) build, but I don't have images of those setups. The adapted code is avaliable in the Piano section.
