# Memory Game
This project was inspired by the memory game Simon. The Arduino generates a random sequence of colors and the lights flash in that order and play a tone corresponding to each color. Then the player has to press buttons and repeat the sequence. If the player repeats the sequence successfully, then a new and slightly longer sequence is generated. If the player misses a color, then the game ends.
